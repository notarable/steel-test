# 🚀 Быстрый старт - Полноэкранное мега-меню

## Что это?

Это готовое решение для создания современного навигационного меню с полноэкранным мега-меню. Идеально подходит для сайтов с большим количеством разделов и страниц.

## ⚡ Быстрый запуск

1. **Откройте файл `index.html`** в браузере (двойной клик)
2. **Нажмите на кнопку "Продукция"** в меню
3. **Мега-меню откроется!** (занимает 85% экрана, снизу видна страница)
4. **Кликните на видимую область снизу** чтобы закрыть меню

## 🎯 Что умеет мега-меню?

### Основное
- ✅ Открывается по клику на кнопку "Продукция"
- ✅ Занимает весь экран (header сверху остаётся)
- ✅ 4 колонки с разными разделами
- ✅ Каждая категория раскрывается при клике
- ✅ Можно открыть несколько категорий одновременно

### Способы закрыть меню
- ✅ **Клик по видимой области страницы снизу** (основной способ)
- ✅ Клавиша **ESC**
- ✅ Клик по любой ссылке внутри меню
- ✅ Клик по пустому пространству меню

## 📝 Как настроить под себя

### 1. Изменить название кнопки

**Файл:** `index.html`  
**Строка:** ~24

```html
<button class="nav-link mega-menu-trigger" id="megaMenuBtn">
    Продукция <!-- Измените здесь -->
    <span class="mega-icon">☰</span>
</button>
```

### 2. Добавить свои разделы

**Файл:** `index.html`  
**Начало:** ~85 (внутри `<div class="mega-menu-grid">`)

**Пример добавления новой категории:**

```html
<div class="mega-category">
    <button class="mega-category-header">
        <span>Моя категория</span>
        <span class="expand-icon">+</span>
    </button>
    <div class="mega-category-content">
        <a href="#link1">📌 Пункт 1</a>
        <a href="#link2">📌 Пункт 2</a>
        <a href="#link3">📌 Пункт 3</a>
    </div>
</div>
```

### 3. Изменить цвета

**Файл:** `style.css`

**Основной цвет меню (синий):**
```css
/* Найдите: #2563eb */
/* Замените на свой цвет, например: #ff6b6b */
```

**Цвет фона мега-меню:**
```css
.mega-menu-overlay {
    background: rgba(0, 0, 0, 0.95); /* Чёрный */
    /* Можно: rgba(255, 255, 255, 0.95) для светлого */
}
```

### 4. Изменить количество колонок

**Файл:** `style.css`  
**Найдите:**

```css
.mega-menu-grid {
    grid-template-columns: repeat(4, 1fr); /* 4 колонки */
}
```

**Варианты:**
- `repeat(2, 1fr)` - 2 колонки
- `repeat(3, 1fr)` - 3 колонки
- `repeat(5, 1fr)` - 5 колонок

### 5. Изменить контакты в футере

**Файл:** `index.html`  
**Найдите блок:** `<div class="mega-menu-footer">`

```html
<div class="mega-footer-item">
    <span class="footer-icon">📞</span>
    <div>
        <strong>Телефон</strong>
        <p>+7 (999) 123-45-67</p> <!-- Ваш телефон -->
    </div>
</div>
```

## 🔧 Типичные задачи

### Убрать кнопку "Продукция" из обычного меню

Если хотите сделать её отдельной кнопкой в header:

```html
<!-- Переместите из <ul class="nav-menu"> сюда: -->
<div class="nav-cta">
    <button class="btn-primary mega-menu-trigger" id="megaMenuBtn">
        Продукция ☰
    </button>
</div>
```

### Закрывать другие категории при открытии новой

**Файл:** `script.js`  
**Найдите и раскомментируйте:**

```javascript
// Опционально: закрываем все другие категории в той же колонке
const column = category.closest('.mega-column');
column.querySelectorAll('.mega-category').forEach(cat => {
    cat.classList.remove('expanded');
});
```

### Добавить поиск в меню

```html
<!-- В начало <div class="mega-menu-content"> -->
<div style="margin-bottom: 40px;">
    <input type="text" 
           placeholder="🔍 Поиск по меню..." 
           style="width: 100%; padding: 15px 20px; font-size: 16px; border: 2px solid rgba(255,255,255,0.2); background: rgba(255,255,255,0.1); color: white; border-radius: 12px;">
</div>
```

## 📱 Адаптивность

Меню автоматически адаптируется под разные устройства:

- **Десктоп (>1024px):** 4 колонки
- **Планшет (768-1024px):** 2 колонки
- **Мобильный (<768px):** 1 колонка

## ❓ Часто задаваемые вопросы

### Как изменить высоту меню?

```css
.mega-menu-overlay {
    height: 85vh; /* Измените на 90vh, 80vh и т.д. */
}

.mega-menu-clickable-area {
    height: 15vh; /* Соответственно измените */
}
```

### Как сделать меню на полный экран?

```css
.mega-menu-overlay {
    height: 100vh; /* Полная высота */
}

/* И скрыть кликабельную область */
.mega-menu-clickable-area {
    display: none;
}
```

### Как изменить скорость анимации?

```css
.mega-menu-overlay {
    transition: all 0.4s ease; /* Измените 0.4s на своё значение */
}
```

### Как добавить иконки к пунктам?

Просто добавьте emoji или Font Awesome иконки:

```html
<a href="#link">🚀 Пункт с иконкой</a>
<a href="#link"><i class="fas fa-rocket"></i> С Font Awesome</a>
```

### Как изменить положение кнопки закрытия?

```css
.mega-menu-close {
    top: 30px;    /* Сверху */
    right: 40px;  /* Справа */
    /* Или слева: left: 40px; right: auto; */
}
```
