# 🎯 Профессиональное навигационное меню с МЕГА-МЕНЮ

Современное, адаптивное меню с полноэкранным мега-меню на чистом HTML, CSS и JavaScript.

## ✨ Возможности

### Основные
- ✅ **Липкое меню** - остаётся вверху при прокрутке
- 🔥 **МЕГА-МЕНЮ** - занимает 85% экрана, снизу виден контент
- ✅ **Закрытие кликом снизу** - интуитивный способ закрытия
- ✅ **Многоуровневая структура** - разделы, подразделы, элементы
- ✅ **Раскрывающиеся категории** - аккордеон для подразделов
- ✅ **Выпадающие подменю** - при наведении курсора (обычное меню)
- ✅ **Плавная прокрутка** - к разделам страницы
- ✅ **Подсветка активного раздела** - автоматически при прокрутке
- ✅ **Анимации** - плавные переходы и эффекты
- ✅ **Современный дизайн** - в стиле 2026 года

### Дополнительные
- 🎨 Красивые hover-эффекты
- 💫 Плавные анимации и backdrop blur
- 📱 Полностью адаптивное
- 🚀 Оптимизированная производительность
- 🎯 CTA кнопка для призыва к действию
- ⌨️ Закрытие по ESC
- 🖱️ Закрытие по клику на фон

## 🚀 Как использовать

1. **Откройте `index.html`** в браузере
2. **Протестируйте мега-меню**:
   - 🔥 Кликните на кнопку **"Продукция"** в меню
   - Откроется меню с 4 колонками (85% экрана)
   - Снизу видна часть страницы с анимированной подсказкой
   - Кликайте на заголовки категорий (Веб-дизайн, Программирование и т.д.)
   - Категории раскрываются, показывая подразделы
   - Видны header сверху и footer снизу
   - **Закройте меню:** кликните на видимую область снизу, ESC, или на ссылку

3. **Протестируйте обычное меню**:
   - Прокрутите страницу вниз - меню останется вверху
   - Наведите курсор на "Услуги" - откроется dropdown подменю
   - Кликните на пункты меню - плавная прокрутка к разделу

## 🎨 Настройка под себя

### Изменить цвета

В файле `style.css` найдите эти переменные:

```css
/* Основной цвет (синий) */
#2563eb → замените на свой цвет

/* Цвет при наведении */
#1d4ed8 → замените на свой цвет

/* Градиент героя */
linear-gradient(135deg, #667eea 0%, #764ba2 100%)
```

### Добавить новый пункт меню

В файле `index.html` в блоке `<ul class="nav-menu">`:

```html
<li class="nav-item">
    <a href="#новый-раздел" class="nav-link">Новый раздел</a>
</li>
```

### Добавить dropdown меню

```html
<li class="nav-item has-dropdown">
    <a href="#раздел" class="nav-link">Раздел <span class="arrow">▼</span></a>
    <ul class="dropdown-menu">
        <li><a href="#подраздел-1">Подраздел 1</a></li>
        <li><a href="#подраздел-2">Подраздел 2</a></li>
    </ul>
</li>
```

### Изменить логотип

В `index.html` найдите:

```html
<div class="logo">
    <a href="#home">МойСайт</a> <!-- Замените на свой текст -->
</div>
```

## 🎯 Настройка мега-меню

### Добавить новую категорию:

В файле `index.html` в блоке `.mega-column`:

```html
<div class="mega-category">
    <button class="mega-category-header">
        <span>Название категории</span>
        <span class="expand-icon">+</span>
    </button>
    <div class="mega-category-content">
        <a href="#link1">🎨 Пункт 1</a>
        <a href="#link2">💻 Пункт 2</a>
        <a href="#link3">🚀 Пункт 3</a>
    </div>
</div>
```

### Добавить новую колонку:

```html
<div class="mega-column">
    <h3 class="mega-title">Заголовок колонки</h3>
    <!-- Категории здесь -->
</div>
```

### Изменить количество колонок:

В файле `style.css` найдите:

```css
.mega-menu-grid {
    grid-template-columns: repeat(4, 1fr); /* Измените 4 на нужное число */
}
```

### Изменить поведение раскрытия:

В файле `script.js` раскомментируйте эти строки, чтобы закрывать другие категории при открытии новой:

```javascript
const column = category.closest('.mega-column');
column.querySelectorAll('.mega-category').forEach(cat => {
    cat.classList.remove('expanded');
});
```
